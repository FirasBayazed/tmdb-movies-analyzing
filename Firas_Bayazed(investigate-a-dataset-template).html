Introduction:¶

This dataset comes from IMDB and contains information about 10,000 movies,short films and tv series collected from The Movie Database (TMDb), including user ratings, revenue, runtime and the budget. In this project, i'll be answering the following questions about the movies:

  • what the genre type of the movie that have the most popularity ,how much budget did it cost,revenue and in which year releaced ?
  • what was the worst movie or tv series,which year released and how much budget did it cost ?
  • did the number of released movies increased per year and in which year released the most numbers of movies?
  • what type of the relationship between budget and revenue ?
  • is there any relationship between budget and runtime ?
  • is there any relationship between revenue and runtime ?
  • what is the relationship between the revenue and the release year ?
  • what type of the relationship between popularity and revenue ?

# this is a function that Udacity reviewers asked for in my first review and I will use it at the end
# function will take the column values,seperate them and count them
def split_count(column_name):
# will take a column, and separate the string by '|'
    column_values = cleaned_movies[column_name].str.cat(sep = '|')
    
# giving pandas series and storing the values separately
    column_values = pd.Series(column_values.split('|'))
    
# this wil give us value in descending order
    count = column_values.value_counts(ascending = False)
    
    return count
